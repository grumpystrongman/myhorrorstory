# QA Strategy

## Test Matrix
- Unit tests for shared packages and domain logic.
- API integration tests for auth, story, party, support.
- Web and admin e2e smoke for critical conversion and gameplay flows.
- Mobile smoke and responsiveness checks.
- Security checks for auth/session/RBAC/rate-limits.

## Certification Artifacts
- UI acceptance storyboards.
- Accessibility check reports.
- Story path validation reports.
- Release smoke logs.
