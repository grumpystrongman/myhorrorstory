# Visual Certification Guide

## Required Captures
- Marketing hero and onboarding flow.
- Case library and case detail view.
- Session play interface including evidence board and clue reveal.
- Mobile home and active session screens.
- Admin dashboard and moderation queue.

## Acceptance Criteria
- Brand palette and typography align with horror-luxe design tokens.
- Text contrast and heading hierarchy pass accessibility checks.
- Layouts validated at desktop, tablet, and mobile breakpoints.
