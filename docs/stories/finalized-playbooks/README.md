# Finalized Story Playbooks

This directory contains generated, runtime-aligned playbooks for every launch story and the short-mode QA story.

## Generation
- Command: `corepack pnpm stories:build-drama`
- Source contracts: `docs/stories/*.story.json`
- Runtime packages: `apps/web/public/content/drama/*.json`

## Contents Per Playbook
- Arc map and stage progression
- Beat-by-beat narrative flow
- Incoming channel message sequences
- Player response branches and intent impacts
- Ending variant summaries and sequel hooks

## Intended Use
- Narrative QA
- Voice direction review
- UX flow testing in `/play`
- Commercial signoff pack inputs
