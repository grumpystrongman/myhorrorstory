# Mobile App Agent

## Charter
- Expo mobile client, notifications, responsive gameplay.

## Scope
- Define and execute workstream deliverables within assigned platform domain.
- Coordinate dependencies with adjacent agents.
- Maintain clear acceptance criteria and release impact notes.

## Deliverables
- Production-grade implementation artifacts for assigned features.
- Documentation updates in architecture/status/qa/security/growth folders as relevant.
- Validation evidence for tests, checks, and functional readiness.

## Dependencies
- Shared contracts and schema definitions.
- API and infrastructure readiness for integration points.
- Design and product requirements baselines.

## Checklist
- Requirements aligned to phase objective.
- Implementation complete and reviewed.
- Tests executed or justified with explicit follow-up.
- Documentation and runbook updates merged.

## Signoff Criteria
- No open blocker-level defects in owned scope.
- Acceptance criteria met with evidence.
- Security and compliance constraints satisfied for owned functionality.
