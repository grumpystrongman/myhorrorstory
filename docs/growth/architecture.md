# Growth and CRM Architecture

## Funnel Stages
- Visitor -> lead capture -> account creation -> first case start -> first case completion -> subscription conversion -> retention loops.

## Lifecycle Automations
- Welcome series.
- Abandoned signup.
- Abandoned case.
- Win-back.
- Upsell and sequel nudges.

Detailed implementation: `docs/growth/email-marketing-system.md`.

## Core KPIs
- Activation rate.
- Day-7 and day-30 retention.
- Case completion rate.
- Trial-to-paid conversion.
- Referral conversion rate.
