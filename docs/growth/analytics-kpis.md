# Analytics Dashboard Definitions

## Product Analytics
- Funnel: Landing -> Signup -> First Case Start -> First Completion -> Paid Conversion.
- Session quality: average chapter completion, clue reveal rate, drop-off chapter.

## Commercial Analytics
- MRR, ARR run-rate, churn, trial conversion, one-time purchase mix.
- Revenue split by story pack and subscription tier.

## Operational Analytics
- Support ticket volume by category and first-response time.
- Admin moderation throughput and unresolved queue.
- Worker queue lag and job failure rates.
